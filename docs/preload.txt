File: preload.js

Purpose: Electron preload script. Exposes secure APIs from the main process to the renderer via contextBridge.

Key Elements:
- Uses contextBridge to expose APIs (electronAPI, wintoolAPI, networkingAPI)
- Provides secure IPC wrappers for:
  - System info, metrics, and tweaks
  - Window controls (minimize, maximize, close)
  - Package management (install, uninstall, get list)
  - System tools and network diagnostics
- No nodeIntegration in renderer

Usage:
- Loaded in BrowserWindow (webPreferences.preload)
- All renderer JS uses these APIs for backend actions

Notes:
- Security: Only exposes whitelisted functions
- No direct Node.js access in renderer context
