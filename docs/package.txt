File: package.json

Purpose: Node.js/Electron project manifest. Defines app metadata, dependencies, scripts, and build configuration for WinTool.

Key Elements:
- name, version, description, author, license
- main: entry point (main.js)
- scripts: start, dist, portable
- build: Electron Builder config (icon, appId, output dir, files)
- dependencies: runtime packages (electron, systeminformation, etc.)
- devDependencies: build and dev tools (electron-builder, tailwindcss, etc.)

Usage:
- Used by npm/yarn for dependency management
- Electron/Node reads for entry point and build config

Notes:
- All files needed for the app must be listed in build.files
- GPL-3.0-or-later license
